Thanks for trying out the extreme alpha of Luno. It's still a work in progress, as you can probably tell - this project aims to take the best concepts from the most popular web frameworks out there (mostly Django, though, because the author is biased as hell). Luno requires Node.js (which you can probably tell if you're reading this by now).

Basic Tutorial...?

I've gone ahead and written a basic tutorial below, to help people get running. With any luck, it'll cause people to contribute patches or something, so here goes. ;D

  1. git clone git://github.com/ryanmcgrath/luno.git
  2. cd luno
  3. sudo node install.js
  4. luno startproject your_project_name
  5. Optionally, also do luno startapp your_app_name
  6. cd your_project_name, configure settings, define urls/views/etc
  7. luno runserver

Seriously, life has never been this cool. ;D
